# portugal-provinces

Function that returns a province that matches the given postal code.

## Install

With [npm](https://npmjs.org) do:

```
npm install portugal-provinces
```

## How to use

``` js
var portugalProvinces = require('provinces')

portugalProvinces('4620-594')
  // => return Porto

```
